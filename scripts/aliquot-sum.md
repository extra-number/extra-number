Get [aliquot sum] of [number].

```javascript
const aliquotSum = require('@extra-number/aliquot-sum');
// aliquotSum(<number>)

aliquotSum(6);
// 6 (1+2+3)
aliquotSum(1);
// 0
aliquotSum(0);
// 0
aliquotSum(-24);
// 36 (1+2+3+4+6+8+12)
```
