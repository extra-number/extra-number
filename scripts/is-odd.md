Check if [number] is [odd].
<br>


```javascript
const isOdd = require('@extra-number/is-odd');

isOdd(23);
// true

isOdd(21.1);
// undefined

isOdd(32);
// false

isOdd(-2);
// false
```

### reference

```javascript
const isOdd = require('@extra-number/is-odd');

isOdd(number)
// number: number to check
// -> true if odd, false if even, undefined otherwise
```
