Count [divisors] of [number].

```javascript
const divisorCount = require('@extra-number/divisor-count');
// divisorCount(<number>)

divisorCount(6);
// 4 (1, 2, 3 ,6)
divisorCount(1);
// 1
divisorCount(0);
// 0
divisorCount(-24);
// 8 (1, 2, 3, 4, 6, 8, 12, 24)
```
