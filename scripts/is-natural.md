Check if [number] is [natural].

```javascript
const isNatural = require('@extra-number/is-natural');
// isNatural(<number>)

isNatural(8);
// true
isNatural(7.1);
// false
isNatural(0);
// false
isNatural(-2);
// false
```

[natural]: https://en.wikipedia.org/wiki/Natural_number
