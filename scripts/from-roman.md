Get [number] from [roman numerals].

```javascript
fromRoman = require('@extra-number/from-roman');
// fromRoman(<string>)

fromRoman('DCXLIX');
// 649
fromRoman('CMXXVIII');
// 928
fromRoman('- x c i x');
// -99
```
