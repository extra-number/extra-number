Check if [number] is [prime].

```javascript
const isPrime = require('@extra-number/is-prime');
// isPrime(<number>)

isPrime(7);
// true
isPrime(53);
// true
isPrime(4);
// false
isPrime(1);
// false
isPrime(0);
// false
```
