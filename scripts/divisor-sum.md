Sum [divisors] of [number].

```javascript
const divisorSum = require('@extra-number/divisor-sum');
// divisorSum(<number>)

divisorSum(6);
// 12 (1+2+3+6)
divisorSum(1);
// 1
divisorSum(0);
// 0
divisorSum(-24);
// 60 (1+2+3+4+6+8+12+24)
```
