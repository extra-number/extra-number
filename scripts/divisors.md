Get [divisors] of [number].

```javascript
const divisors = require('@extra-number/divisors');
// divisors(<number>)

divisors(6);
// [1, 2, 3, 6]
divisors(1);
// [1]
divisors(0);
// []
divisors(-24);
// [1, 2, 3, 4, 6, 8, 12, 24]
```
