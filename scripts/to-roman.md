Convert [number] to [roman numerals].

```javascript
const toRoman = require('@extra-number/to-roman');
// toRoman(<number>)

toRoman(95);
// 'XCV'
toRoman(1889);
// 'MDCCCLXXXIX'
toRoman(468);
// 'CDLXVIII'
toRoman(-943);
// '-CMXLIII'
```
